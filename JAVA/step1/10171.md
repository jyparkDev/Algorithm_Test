# 아래 예제와 같이 고양이를 출력하시오.
```
\    /\
 )  ( ')
(  /  )
 \(__)|
 ```

 ```java
import java.lang.*;
public class Main{
    public static void main(String[] args){
        System.out.println("\\    /\\");
        System.out.println(" )  ( \')");
        System.out.println("(  /  )");
        System.out.println(" \\(__)|");
    }
}
 ```
